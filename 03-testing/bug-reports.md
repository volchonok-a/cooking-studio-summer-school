# Отчёты о багах

Проект: Кулинарная Студия "Шеф-стол"  
Дата: 06.07.2026

## BUG-01: Отсутствие валидации email

| Параметр | Значение |
|----------|----------|
| Серьёзность | Средняя |
| Приоритет | Средний |
| Статус | Исправлен |

**Описание:** В модальном окне входа пользователь мог ввести некорректный email (например, "abc") и форма отправлялась без проверки.

**Шаги воспроизведения:**
1. Открыть главную страницу
2. Нажать "Войти в аккаунт"
3. Ввести email: "invalid-email"
4. Ввести пароль: "123456"
5. Нажать "Войти"

**Фактический результат:** Форма отправлялась без проверки email.

**Ожидаемый результат:** Появляется сообщение "Введите корректный email". Форма не отправляется.

**Исправление:** Добавлена функция validateEmail() с регулярным выражением /^[^\s@]+@[^\s@]+\.[^\s@]+$/.

## BUG-02: Отсутствие проверки длины пароля

| Параметр | Значение |
|----------|----------|
| Серьёзность | Средняя |
| Приоритет | Средний |
| Статус | Исправлен |

**Описание:** В модальном окне входа пользователь мог ввести пароль из 1-2 символов.

**Шаги воспроизведения:**
1. Открыть модальное окно входа
2. Ввести корректный email
3. Ввести пароль: "12"
4. Нажать "Войти"

**Фактический результат:** Форма отправлялась с коротким паролем.

**Ожидаемый результат:** Появляется сообщение "Пароль должен быть не менее 6 символов". Форма не отправляется.

**Исправление:** Добавлена функция validatePassword(), проверяющая password.length < 6.

## BUG-03: Отсутствие выбора экипировки по умолчанию

| Параметр | Значение |
|----------|----------|
| Серьёзность | Низкая |
| Приоритет | Низкий |
| Статус | Исправлен |

**Описание:** Если пользователь не выбирал экипировку, форма могла отправиться с некорректными данными.

**Шаги воспроизведения:**
1. Открыть экран бронирования
2. Не выбирать тип экипировки
3. Заполнить остальные поля
4. Нажать "Подтвердить бронь"

**Фактический результат:** Форма отправлялась без выбранного типа экипировки.

**Ожидаемый результат:** По умолчанию выбрана "Своя экипировка".

**Исправление:** Добавлен атрибут checked к радиокнопке "Своя экипировка".
