# Фича 3: Ограничение максимального количества мест

## 1. Симптом / Цель
**Симптом:** В поле "Количество мест" можно ввести любое число, включая отрицательные и нереалистично большие значения. Кулинарная студия имеет физическое ограничение по количеству посадочных мест.

**Цель:** Ограничить ввод допустимым диапазоном (1-12 мест), чтобы предотвратить некорректные бронирования.

## 2. Требования
- **FR-13:** Количество мест должно быть в разумных пределах
- **BUS-01:** Максимальная вместимость студии — 12 человек
- **UX-05:** При вводе недопустимого значения пользователь должен увидеть понятное сообщение

## 3. Реализация
Добавлены ограничения на поле "Количество мест":
- HTML-атрибуты min="1" и max="12" для числового поля
- JavaScript-валидация при изменении значения
- Если введено значение вне диапазона — показывается сообщение "Доступно от 1 до 12 мест"

## 4. Используемые инструменты
- **ИИ:** Qwen (помощь в разработке ограничений)
- **Среда:** VS Code
- **Технологии:** HTML5 (input type=number с min/max), JavaScript

## 5. Проверка вручную
1. Открыть index.html в браузере
2. Ввести 0 или отрицательное число -> должно появиться сообщение об ошибке
3. Ввести 15 или больше -> должно появиться сообщение об ошибке
4. Ввести 5 -> должно работать корректно
5. Использовать стрелки вверх/вниз -> значение должно меняться в пределах 1-12
