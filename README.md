# Кулинарная Студия "Шеф-стол"

Проект летней школы Surf Studio 2026. Веб-приложение для записи на мастер-классы кулинарной студии.

## Стек технологий

**Backend:**
- Go
- PostgreSQL
- Docker, Docker Compose
- REST API
- Goose (миграции)
- K6 (нагрузочное тестирование)

**Frontend:**
- HTML5, CSS3, JavaScript
- Fetch API

## Структура проекта

```
cooking-studio-summer-school/
├── 01-analytics/          # Аналитика (требования, макеты)
├── 02-development/        # Документация по разработке
│   ├── feature-1-.../     # Фичи (3 шт.)
│   └── bug-1-.../         # Исправленные баги (3 шт.)
├── 03-testing/            # Тест-кейсы и отчёты
├── backend/               # Backend на Go
└── web-client/                # Frontend (HTML/CSS/JS)
```

## Как запустить

### Backend

```bash
cd backend
docker compose --profile db up -d db
docker compose --profile migrations run --rm migrate
docker compose --profile app up --build
```

Backend доступен на `http://127.0.0.1:8080`

Эндпоинты:
- `GET /healthz` — проверка здоровья
- `GET /slots` — список мастер-классов
- `POST /bookings` — создание брони

### Frontend

Открыть `web-client/index.html` в браузере. Backend должен быть запущен.
