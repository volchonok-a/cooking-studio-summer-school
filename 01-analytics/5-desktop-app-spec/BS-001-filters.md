# BS-001: Фильтры слотов

**ID:** BS-001

**Тип:** Бизнес-сценарий / Паттерн

**Домен:** 02. Просмотр слотов

**Приоритет:** High

**Статус:** Актуален

---

## Описание
Клиент фильтрует список доступных слотов по различным параметрам (дата, программа, шеф, наличие мест) для быстрого поиска подходящего класса.

### Цель
Уменьшить время поиска нужного слота и повысить конверсию в бронирование за счёт релевантной выдачи.

### Контекст использования
Применяется на экране SCR-002 (Список слотов).

---

## Элементы интерфейса

### Панель фильтров (Модалка / Выпадающий список)

| Параметр | Значение |
|----------|----------|
| Label | «Диапазон дат» |
| Тип | Date picker (от — до) |
| Значения | Любые валидные даты |
| По умолчанию | [сегодня; сегодня + 7 дней] |

| Параметр | Значение |
|----------|----------|
| Label | «Программа» |
| Тип | Dropdown или чекбоксы |
| Значения | Опции из справочника программ |
| По умолчанию | Все программы |

| Параметр | Значение |
|----------|----------|
| Label | «Шеф» |
| Тип | Dropdown или чекбоксы |
| Значения | Имена шефов |
| По умолчанию | Все шефы |

| Параметр | Значение |
|----------|----------|
| Label | «Наличие мест» |
| Тип | Checkbox |
| Значения | true/false |
| По умолчанию | false (показывать все) |

### Кнопки управления

| Параметр | Значение |
|----------|----------|
| Основная | «Применить» |
| Вторичная | «Сбросить» |
| Третичная | «Отмена» (закрытие без применения) |

---

## Сценарии / Состояния

### Сценарий 1: Активные фильтры
**Контекст:** SCR-002 после применения фильтров.
**Элемент:** Теги над списком слотов.
**Текст:** «Программа: Итальянская кухня ×», «Шеф: Иван Петров ×», ссылка «Сбросить все».
**Действие:** Клик по крестику удаляет фильтр, клик по «Сбросить все» очищает всё.

### Сценарий 2: Нет результатов
**Контекст:** SCR-002, когда фильтры не дали совпадений.
**Элемент:** Empty state.
**Текст:** «По вашему запросу ничего не найдено».
**Действие:** Кнопка «Сбросить фильтры».

---

## Бизнес-правила

- Фильтры комбинируются по **AND-логике** (должны выполняться все условия).
- При нажатии «Применить» — автоматический запрос к API `/slots` с query-параметрами.
- Сброс фильтров возвращает дефолтный диапазон (7 дней).
