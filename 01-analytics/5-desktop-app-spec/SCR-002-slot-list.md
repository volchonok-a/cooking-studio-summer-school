# SCR-002: Список слотов

## Описание
Экран отображает доступные для записи мастер-классы на ближайшие 7 дней.

## Элементы интерфейса
- **Заголовок:** «Кулинарные классы»
- **Панель фильтров:** Кнопка [Фильтры] (открывает `BS-001`)
- **Список слотов:**
    - Группировка: по датам (Сегодня, Завтра, и т.д.)
    - Карточка слота:
        - Дата/время: [DD.MM HH:mm]
        - Программа: [Название]
        - Шеф: [Имя] + рейтинг [★]
        - Цена: [Цена] ₽
        - Места: [N из M] мест свободно

## Состояния
- **Loading:** Skeleton-экраны для карточек.
- **Empty State:** [Иконка календаря] «Пока нет доступных классов». Кнопка [Расширить диапазон].
- **Error State:** Баннер с кнопкой «Повторить» (согласно `BS-003`).
