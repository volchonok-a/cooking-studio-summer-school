# BS-003: Состояния ошибок (Error States)

**ID:** BS-003

**Тип:** Состояние интерфейса / Паттерн

**Домен:** Общий (UX)

**Приоритет:** High

**Статус:** Актуален

---

## Описание
Единые паттерны отображения ошибок в клиентском приложении. Все ошибки делятся на системные (сеть, таймауты), валидации и бизнес-логики. Сообщения должны быть понятными, не техническими и предлагать действие для исправления.

### Цель
Сделать обработку ошибок прозрачной для пользователя и снизить количество обращений в поддержку.

### Контекст использования
Все экраны приложения (SCR-001...SCR-010).

---

## Элементы интерфейса

| Тип ошибки | UI-элемент | Где применяется |
|------------|------------|-----------------|
| Inline | Красный текст под инпутом | Ошибки валидации форм |
| Banner | Жёлтая/красная плашка на всю ширину | Ошибки загрузки, офлайн, блокировка |
| Toast | Небольшое уведомление в углу | Успешные действия, короткие ошибки |
| Modal | Центрированное окно с затемнением | Критические предупреждения |
| Empty/Error | Иконка + текст + кнопка | Полный экран при ошибке загрузки |

---

## Сценарии / Состояния

### Сценарий 1: Ошибки валидации
**Контекст:** SCR-001, SCR-004.
**Текст:** «Введите корректный Email» / «Укажите аллергии или "Нет / не беспокоят"».
**Действие:** Кнопка отправки формы остаётся `disabled`.

### Сценарий 2: Нет мест / Гонка за места
**Контекст:** SCR-004.
**Текст:** «Места закончились. К сожалению, пока вы заполняли форму, места заняли. Выберите другой слот».
**Действие:** Кнопка «Вернуться к списку» (редирект на SCR-002).

### Сценарий 3: Сетевые ошибки и таймауты
**Контекст:** Все экраны.
**Текст:** «Превышено время ожидания ответа сервера» / «Нет подключения к интернету».
**Действие:** Кнопка «Повторить». Мутации запрещены.

---

## Бизнес-правила

- **Не обвинять пользователя.** Вместо «Вы не заполнили поле» → «Пожалуйста, укажите аллергии».
- **Не показывать технические коды.** Вместо «Error 500» → «Что-то пошло не так. Попробуйте ещё раз».
- **Предлагать решение.** Каждая ошибка заканчивается понятным действием.
- Язык: Русский.
