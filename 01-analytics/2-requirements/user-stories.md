# Этап 2. Пользовательские истории

Формат: «Как <роль>, я хочу <действие>, чтобы <ценность>». Каждой истории — критерии приёмки.

**Скоуп:** клиентское веб-приложение (десктоп) и API для него (роль «Клиент»). Функции шефа и владельца обеспечиваются существующей инфраструктурой и здесь не описываются.

| ID | Как (роль) | Хочу (действие) | Чтобы (ценность) | Критерии приёмки | FR |
| --- | --- | --- | --- | --- | --- |
| US-1 | Клиент | зарегистрироваться или войти через Email+пароль либо OAuth (Google/Яндекс) | получить доступ к личному кабинету для записи на классы | Поддержаны Email+пароль и OAuth (Google, Яндекс). SMS-авторизация исключена для MVP. После успешного входа — редирект на главный экран со слотами. При ошибке — понятное сообщение. | FR-01, FR-02 |
| US-2 | Клиент | видеть в профиле счетчик посещений и статус «Постоянный клиент» | понимать свою лояльность и получать бонусы | При >5 посещениях отображается бейдж «Постоянный клиент» и автоматически применяется скидка 5% при бронировании. Счетчик обновляется после завершения класса (данные с бэкенда). | FR-03, FR-04 |
| US-3 | Клиент | видеть список предстоящих слотов на ближайшие 7 дней | быстро выбрать удобное время для записи | По умолчанию показываются слоты на 7 дней от текущего момента, сгруппированные по дате и отсортированные по времени. Слот показывает дату/время, программу, шефа, цену, всего/свободно мест. При отсутствии расписания — empty state «Пока нет доступных классов». | FR-05, FR-08 |
| US-4 | Клиент | фильтровать список слотов | быстро находить подходящий класс | Доступны фильтры: диапазон дат (расширение за пределы 7 дней), программа, шеф, наличие мест. Фильтры комбинируются. При отсутствии результатов — empty state. | FR-06 |
| US-5 | Клиент | открыть детальную карточку слота | изучить все параметры перед записью | Карточка показывает программу/меню, длительность (~3 ч), шефа (имя, рейтинг), стоимость, стоимость проката экипировки, число свободных мест, адрес студии. Кнопка «Записаться» активна только при наличии мест. | FR-07 |
| US-6 | Клиент | записаться на выбранный слот | забронировать место на кулинарном классе | После подтверждения создается запись (бронь). Бэкенд атомарно проверяет наличие мест (защита от овербукинга и двойных броней). При успехе — экран подтверждения, запись появляется в «Моих бронированиях». | FR-09, FR-10 |
| US-7 | Клиент | при записи указать свою или прокатную экипировку | студия подготовила нужный инвентарь | На форме брони переключатель: «Своя экипировка» / «Прокат (фартук + ножи)». При выборе проката показывается доп. плата, она добавляется к итоговой сумме. Лимита на прокатные наборы нет. | FR-11, FR-12 |
| US-8 | Клиент | заполнить обязательное поле «Аллергии и особенности питания» | шеф заранее адаптировал меню и обеспечил безопасность | Поле обязательно для заполнения. При пустом поле кнопка «Записаться» неактивна, показывается подсказка. Допустимо указать «Нет / не беспокоят». | FR-12 |
| US-9 | Клиент | одной записью забронировать несколько мест (себя + гости) | записать компанию за раз без повторных оформлений | Поле «Количество мест» (от 1 до числа свободных). Сумма пересчитывается с учетом количества мест и скидки 5% (если применима). Выбор экипировки в MVP — единый на всю бронь. | FR-13, FR-14 |
| US-10 | Клиент | видеть итоговую стоимость до подтверждения брони | избежать сюрпризов при оплате на месте | Показаны: базовая цена × кол-во мест, доплата за прокат, скидка 5% (если постоянный клиент), итого. Подпись: «Оплата на месте (наличные / перевод на карту)». Онлайн-оплата исключена. | FR-14, FR-15, FR-16 |
| US-11 | Клиент | видеть список своих бронирований | контролировать предстоящие и прошедшие классы | Вкладки/фильтры: «Предстоящие», «Прошедшие», «Отменённые». Для каждой брони: дата/время, программа, шеф, статус, число мест, экипировка, аллергии, кнопка действия (отменить / оценить). | FR-18 |
| US-12 | Клиент | отменить свою запись до старта | освободить место, если планы изменились | Ранняя отмена (≥12 часов до старта): статус «Отменено клиентом», место освобождается. Поздняя отмена (<12 часов): предупреждение о блокировке на 7 дней. После подтверждения — бронь отменена, клиент заблокирован (на стороне бэкенда). Во время блокировки запись на классы недоступна. | FR-19, FR-20, FR-21 |
| US-13 | Клиент | узнать об отмене класса студией (форс-мажор) | не приехать зря и перепланировать вечер | Если класс отменен студией, бронь не удаляется, получает статус «Отменён студией» + причина. Клиент получает push-уведомление. Повторная запись на этот отменённый слот запрещена. | FR-22, FR-26 |
| US-14 | Клиент | поставить шефу оценку (1–5 звёзд) после посещения | оставить обратную связь для студии и других клиентов | Оценка доступна только для прошедших классов со статусом «Состоялся» (один раз на бронь). Текстовые комментарии в MVP не реализуются. После отправки — обновление рейтинга шефа. | FR-23, FR-24 |
| US-15 | Клиент | получать системные push-уведомления | не забыть про запись и узнать о важных изменениях | Push-напоминание за ~24 часа до старта. Push при отмене класса студией (с причиной). SMS/Email не реализуются. Запрос разрешения на push при первом входе/брони. | FR-25, FR-26, FR-27 |
| US-16 | Клиент | видеть внутриприложенные уведомления (колокольчик) | не пропустить важные события даже без push | Иконка колокольчика в шапке с бейджем непрочитанных. Внутри — список событий: подтверждение брони, отмена клиентом, отмена студией, окончание блокировки. | FR-28 |
| US-17 | Клиент | видеть статус блокировки при нарушении правил отмены или no-show | понимать, когда смогу снова записаться | При попытке забронировать во время блокировки (7 дней после поздней отмены или неявки) — сообщение «Записи недоступны до <дата>». Кнопки бронирования неактивны. | FR-29, FR-30 |
