# BS-001: Фильтры слотов

## Описание
Клиент фильтрует список слотов по различным параметрам.

## Элементы интерфейса (панель/модалка)

### Фильтр по дате
- Label: **«Диапазон дат»**
- Date picker (от — до)
- По умолчанию: [сегодня; сегодня + 7 дней]

### Фильтр по программе
- Label: **«Программа»**
- Dropdown или чекбоксы
- Опции из справочника программ

### Фильтр по шефу
- Label: **«Шеф»**
- Dropdown или чекбоксы
- Опции: имена шефов

### Фильтр по наличию мест
- Label: **«Наличие мест»**
- Checkbox: «Показывать только со свободными местами»

### Кнопки
- **«Применить»** (основная)
- **«Сбросить»** (вторичная)
- **«Отмена»** (закрытие без применения)

## Состояния

### 1. Активные фильтры
- Отображаются как теги над списком:
  - «Программа: Итальянская кухня ×»
  - «Шеф: Иван Петров ×»
  - «Сбросить все»

### 2. Нет результатов
- Empty state: **«По вашему запросу ничего не найдено»**
- Кнопка **«Сбросить фильтры»**

## Бизнес-правила
- Фильтры комбинируются (AND-логика)
- При изменении фильтров — автоматический запрос к API
