# BS-002: Пустые состояния

## Описание
Дизайн empty states для всех списков и разделов.

## Сценарии

### 1. Нет слотов вообще
**Контекст:** SCR-002 (список слотов)  
**Иконка:** календарь или пустая тарелка  
**Заголовок:** «Пока нет доступных классов»  
**Текст:** «Попробуйте расширить диапазон дат или зайдите позже»  
**Действие:** кнопка «Расширить диапазон»

### 2. Нет результатов по фильтру
**Контекст:** SCR-002 + BS-001  
**Иконка:** фильтр или лупа  
**Заголовок:** «По вашему запросу ничего не найдено»  
**Текст:** «Попробуйте изменить фильтры»  
**Действие:** кнопка «Сбросить фильтры»

### 3. Нет бронирований
**Контекст:** SCR-006 (мои бронирования)  
**Иконка:** пустой список  
**Заголовок:** «У вас пока нет бронирований»  
**Текст:** «Запишитесь на первый класс!»  
**Действие:** кнопка «Перейти к классам»

### 4. Нет бронирований в категории
**Контекст:** SCR-006, вкладка «Прошедшие»  
**Иконка:** часы  
**Заголовок:** «У вас пока нет прошедших классов»  
**Текст:** «—»  
**Действие:** нет

### 5. Нет уведомлений
**Контекст:** SCR-010 (колокольчик)  
**Иконка:** колокольчик  
**Заголовок:** «Нет новых уведомлений»  
**Текст:** «—»  
**Действие:** нет

## Принципы
- Иконка (визуальный якорь)
- Заголовок (что произошло)
- Текст (почему и что делать)
- Действие (кнопка, если уместно)

## Связи
- **FR:** FR-08
- **NFR:** NFR-1 (понятные empty states)
