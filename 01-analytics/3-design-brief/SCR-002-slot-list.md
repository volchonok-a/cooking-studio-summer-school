# SCR-002: Список слотов

## Описание
Клиент видит список доступных слотов на ближайшие 7 дней, сгруппированных по дате.

## Элементы интерфейса

### Заголовок
- **«Кулинарные классы»** или **«Ближайшие классы»**

### Панель фильтров
- Кнопка **«Фильтры»** (открывает BS-001)
- Отображение активных фильтров (теги с возможностью сброса)

### Список слотов
Группировка по дате:
- **Заголовок группы:** «Сегодня», «Завтра», «Пн, 10 июля» и т.д.
- **Карточка слота** (переход на SCR-003):
  - Дата и время
  - Название программы
  - Имя шефа + рейтинг (звёзды)
  - Цена
  - Индикатор мест: «5 из 12 мест свободно»

### Кнопка «Показать ещё»
- Расширение диапазона дат за пределы 7 дней

## Состояния

### 1. Загрузка
- Skeleton-экран или spinner

### 2. Empty state (нет слотов вообще)
- Иконка (календарь/пустая тарелка)
- Текст: **«Пока нет доступных классов»**
- Подтекст: **«Попробуйте расширить диапазон дат»**
- Кнопка **«Расширить диапазон»**

### 3. Empty state (нет по фильтру)
- Текст: **«По вашему запросу ничего не найдено»**
- Подтекст: **«Попробуйте изменить фильтры»**
- Кнопка **«Сбросить фильтры»**

### 4. Ошибка загрузки
- Текст: **«Не удалось загрузить расписание»**
- Кнопка **«Повторить»**

### 5. Офлайн (данные из кэша)
- Плашка сверху: **«Данные могли устареть»**
- Список отображается из кэша

## Бизнес-правила
- По умолчанию: слоты на 7 дней от текущего момента
- Группировка по дате, сортировка по времени внутри группы
- Показывать только слоты с свободными местами (или все, но с индикатором)

## Связи
- **FR:** FR-05, FR-08
- **US:** US-3
- **NFR:** NFR-4 (p95 < 2.0 с)
